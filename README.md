<u>**Automated-Rolimons-Trade-Ad-Creator**<u>
 - Automatically creates trade ads on Rolimons at user configured intervals.

<u>User Guide<u>
 - The program will first query user config, starting with player id selection. Copy the id located in your url tab.
   
<img width="387" height="31" alt="image" src="https://github.com/user-attachments/assets/b023f46a-273a-4442-bd57-1e9a3ab0c266" />

 - The program will then ask for your offer item ids. Separate the ids with commas, you can offer a maximum of four items.
 - The program will then ask for request item ids. Once again, separate with commas, however if you only want to use request tags and do not want to offer an item, input a 0.
 - The program will ask for request tags, separate these values with commas.
 - The program will ask for your _RoliVerification cookie. To access your cookie press F12 on any Rolimons page and go to storage, underneath cookies you will find a tab labelled _RoliVerification (Beneath your _RoliData cookie). Copy that cookie and paste it exactly into the box. No cookies are stored or sent, however are necessary to create a trade ad.

<img width="506" height="197" alt="image" src="https://github.com/user-attachments/assets/06b2d4d8-6a67-44ac-b8c8-3fccc0364167" />

- Lastly, the program will ask for the wait duration (in seconds). I recommend either 15 minutes (900 seconds) if you want to send trades as fast as possible, or 24 minutes if you want to space out your trade ads throughout the day (otherwise you'll reach the daily limit, 24 minutes is the minimum time without reaching this limit). I also recommend a small buffer period, to avoid going out of sync if you have a poor internet connection (3-10 seconds will work).
